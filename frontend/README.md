# 정부지원사업 프론트엔드

테크 뉴스레터 스타일의 모던한 UI로 정부지원사업 공고를 확인하세요.

## 🚀 실행 방법

### 1. 백엔드 API 서버 실행 (필수)
```bash
# 프로젝트 루트에서
venv\Scripts\python.exe -m app.main
```

### 2. 프론트엔드 실행

#### 방법 A: 직접 파일 열기
```bash
# frontend 디렉토리의 index.html을 브라우저로 열기
start frontend\index.html
```

#### 방법 B: 간단한 웹 서버 사용 (권장)
```bash
# Python 내장 서버 사용
cd frontend
python -m http.server 3000

# 브라우저에서 접속
http://localhost:3000
```

## ✨ 주요 기능

### 실시간 데이터
- **통계 대시보드**: 오늘 신규 / 진행중 / 마감임박 / 전체 공고 수
- **자동 업데이트**: 5분마다 최신 데이터 자동 갱신

### 섹션별 공고
1. **오늘의 신규 공고**: 최근 추가된 공고 (NEW 배지)
2. **진행중인 공고**: 현재 신청 가능한 모든 공고
3. **마감 임박**: 7일 이내 마감 공고 (우선 확인)

### 스마트 표시
- **소스 배지**: 과기부 / K-Startup 구분
- **마감일 표시**: D-day 카운트다운
- **카테고리 태그**: 빠른 분류 확인
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화

## 🎨 디자인 특징

- **클린한 화이트 배경**: 가독성 최우선
- **터키쿼이즈 액센트**: 현대적이고 신선한 느낌
- **카드 기반 레이아웃**: 스캔하기 쉬운 구조
- **호버 애니메이션**: 부드러운 인터랙션

## 🔗 API 연동

모든 데이터는 백엔드 API에서 실시간으로 가져옵니다:

```javascript
API 엔드포인트:
```

## 🛠️ 트러블슈팅

### CORS 에러 발생 시
백엔드 `app/main.py`에서 CORS 설정 확인:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 이미 설정됨
    ...
)
```

### 데이터가 안 보이는 경우
1. 백엔드 API 서버가 실행중인지 확인
2. 크롤러로 데이터 수집 확인: `python minimal_test.py`
3. 브라우저 콘솔(F12)에서 에러 확인
