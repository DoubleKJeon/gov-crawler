## 🚀 즉시 사용 가능

```bash
# 1. 서버 실행
run_server.bat

# 2. Swagger UI 접속
http://localhost:8000/docs

# 3. API 테스트
- 회원가입 → 로그인 → 토큰 획득
- Authorize 버튼 클릭 → 토큰 입력
- 모든 API 테스트!
```

## 🌟 특별한 점

### 1. 완전 자동화

- 매일 새로운 공고 수집
